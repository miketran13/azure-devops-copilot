# DevOps Copilot — AI-Powered Azure DevOps Extension

Bring AI-powered copilot capabilities directly into Azure Boards.

## Features

- **AI Chat Hub** — Natural language interface under Azure Boards for work item management
- **Work Item AI Panel** — Contextual AI analysis on every work item form
- **Smart Context Menus** — Right-click actions for AI analysis, test case generation, and more
- **Multi-Agent Architecture** — Specialist agents for search, writing, and analysis coordinated by an intelligent orchestrator

## Getting Started

Install the extension from the Visual Studio Marketplace, configure your backend API endpoint, and start chatting with your Azure DevOps backlog.

See the [GitHub repository](https://github.com/YOUR-ORG/devops-copilot) for setup instructions and documentation.
